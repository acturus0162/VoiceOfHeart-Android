# VoiceOfHeart-Android

## 작업 규칙

- 화면은 `ui` 패키지 안에 새 패키지를 만들어서 작업합니다. 예: `ui/login`, `ui/home`
- 각 화면 작업으로 발생하는 기타 파일들은 모두 해당 화면의 패키지 내부에서 생성합니다.
- `component` 패키지에는 앱의 여러 화면에 공통으로 쓰이는 컴포넌트를 둡니다.
- 공통 컴포넌트 파일 생성은 자유롭게 해도 되지만, 기존 컴포넌트를 수정하는 경우 패키지 관리자 1명이 수정합니다.
- 마찬가지로 `theme` 패키지의 `Color`, `Type`, `Theme` 파일들도 패키지 관리자 1명이 관리합니다.
